# Weather Hunters Dashboard

This is a React-based dashboard for monitoring and running the Weather Hunters program. It is designed to be deployed on Replit.

## Features
- Monitor program status
- Run Weather Hunters scripts
- User-friendly interface

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```

## Deployment
This project is configured to run on Replit. Simply open the project in Replit and it will automatically start the dashboard.

## File Structure
- `src/`: Contains the React components and styles.
- `public/`: Contains the HTML template.
- `.replit` and `replit.nix`: Configuration files for Replit.
